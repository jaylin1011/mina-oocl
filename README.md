# mina-oocl

一个基于原生 **MINA** 框架和 **WeUI** 库开发的小程序（**OOCL** 主题）
a mini program with an oocl theme based on wechat native MINA and WeUI

## Start

```bash
npm install

# 在微信开发者工具中构建 npm
```

> Tips：因为本项目是为了展示使用，且因为借用相关公司主题并未正式部署和发布，您可能没有相关权限获取到云开发中使用的数据而无法正常展示，所以可以根据下面图片参考，自行提供数据修改。
